# Diaa's Portfolio — Starter Site

React + TypeScript + Tailwind CSS v4 portfolio, ready to customize and deploy free.

## Run it locally

```bash
cd portfolio
npm install
npm run dev
```

Open the URL it prints (usually http://localhost:5173).

## What to edit

1. **src/data/projects.ts** — replace the 4 placeholder projects with your real
   project write-ups (problem / approach / result / tools / GitHub link).
2. **src/components/About.tsx** — replace the bio paragraph and the
   education/experience rows.
3. **src/components/Contact.tsx** — replace email, GitHub, and LinkedIn links.
4. **src/components/Navbar.tsx** — put your real resume PDF at `public/resume.pdf`
   (the button already links to `/resume.pdf`).

## Deploy free (Vercel)

1. Push this folder to a new GitHub repo.
2. Go to vercel.com, sign in with GitHub, "Add New Project", pick the repo.
3. Vercel auto-detects Vite — just click Deploy.
4. You'll get a free live URL like `your-repo.vercel.app`.
   Every push to GitHub auto-redeploys.

(Netlify works the same way if you prefer it.)
